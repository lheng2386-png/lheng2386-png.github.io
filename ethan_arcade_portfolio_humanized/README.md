# Ethan Lee Manga Arcade Portfolio Ultra Edition

这是 GitHub Pages 个人主页文件夹版本。

## 文件结构

```text
ethan_arcade_portfolio_ultra/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
└── README.md
```

## 使用方法

把整个文件夹里的内容上传到你的 GitHub Pages 仓库根目录即可。

也就是仓库根目录最终应该是：

```text
index.html
css/style.css
js/main.js
README.md
```
